EnterTheMobile
==============

Im Wesentlichen dient dieses kleine Projekt dazu, einige latente Kenntnisse mal wieder
zu erwecken. Entstanden ist ein trivialer Einkaufshelfer aus zwei Teilen - als
WebApp.

Der Offline Bereich verwaltet eine Liste von Produkten, die man einkaufen muss 
und eine Liste von Märkten, bei denen man genau das tun kann. Nichts besonderes
soweit, das gibt es als App wie Sand am Meer.

Online sorgt eine MySql Datenbank mit ganz wenig PHP Code dafür, dass man zumindest
die Liste der Produkte nicht nur Online verwalten sondern auch zwischen 
mehreren Geräten austauschen kann. So kann ich am PC oder an einem SmartPhone
die Liste pflegen und mit einem anderen SmartPhone einkaufen gehen.

Alles in allem sehr rudimentär, ich schaue mir das mal in der Praxis an - vielleicht
wird ja was richtiges daraus. Immerhin ist die Datenbank wie man so schön sagt
schon 'mandantenfähig': mit einem Anmeldecode können verschiedene Listen
unterschiedlicher Benutzer separat gepflegt werden. Die Sicherheit ist zwar
eher übersichtlich, aber das ist ja auch nur ein Spielzeug zum Lernen.
